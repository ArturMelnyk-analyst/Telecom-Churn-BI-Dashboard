\# Telecom Churn BI Dashboard



Business Intelligence project based on the IBM Telco Customer Churn dataset.



\## Goal



Use Python (pandas) and Tableau to:



\- Understand telecom customer churn behavior

\- Quantify revenue at risk

\- Identify high-risk customer segments

\- Propose actionable retention strategies



\## Tools



\- \*\*Python\*\* (pandas, Jupyter / Colab) for data loading, cleaning and feature engineering  

\- \*\*Tableau\*\* for interactive dashboards and business insights



\## Data



\- IBM Telco Customer Churn sample dataset  

\- Original file: `data/raw/Telco\_customer\_churn.xlsx` (added in a later PR)
